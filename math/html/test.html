Последовательности

Обозначение

\(a_1,~a_2,~a_3,\ldots ,a_n\)
  • \(a_i\) — элемент последовательности с номером \(i\).
  • \(a_1 \leq a_2 \leq \ldots \leq a_n\) — если нет речи про порядок.
  • \(a_1 < a_2 < \ldots < a_n\) — если нет речи про порядок и числа различные.
  • \(d_i = a_{i+1} - a_i\) — разность соседних элементов.

Оценка суммы последовательности необязательно различных чисел

\(n \cdot \underset{i}{\operatorname{min}}{(a_i)} \leq a_1 + a_2 + \ldots + a_n \leq n \cdot \underset{i}{\operatorname{max}}{(a_i)}\).

Оценка суммы последовательности различных чисел

  • Если все \(a_i \in \{ b_1,~b_2,~b_3,\ldots \},~~b_1 < b_2 < b_3 < \ldots\), то
    \(a_1 + a_2 + \ldots + a_n \geq b_1 + b_2 + \ldots + b_n\).
  • Если все \(a_i \in \{ b_1,~b_2,~b_3,\ldots \},~~b_1 > b_2 > b_3 > \ldots\), то
    \(a_1 + a_2 + \ldots + a_n \leq b_1 + b_2 + \ldots + b_n\).
