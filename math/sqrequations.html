Общего вида

Уравнение:

\(ax^2+bx+c = 0,~~a \neq 0\).

Решение:

\(D = b^2 - 4ac = d^2,\)

\(\union{ x = \displaystyle \frac{-b + d}{2a} \\ x = \displaystyle \frac{-b - d}{2a} }. \)

Пример:

\(-2x^2 + 3x+4 = 0 ,\)

\(D = 3^2+4\cdot 2\cdot 4 = 32 = \left(4\sqrt{2}\right)^2,\)

\( \union{ x = \displaystyle \frac{-3 + 4\sqrt{2}}{-4} \\ x = \displaystyle \frac{-3 - 4\sqrt{2}}{-4} }. \)

Второй чётный коэффициент

Уравнение:

\(ax^2+bx+c = 0,~~a \neq 0,~~b ~\vdots~ 2.\)

Решение:

\(\displaystyle\frac{D}{4} = \left(\frac{b}{2}\right)^2 - ac = d^2,\)

\( \union{ x = \displaystyle \frac{-\frac{b}{2} + d}{a} \\ x = \displaystyle \frac{-\frac{b}{2} - d}{a} }. \)

Пример:

\(3x^2 -4x-5 = 0 ,\)

\(\displaystyle\frac{D}{4} = 2^2+3\cdot 5 = 19 = \left(\sqrt{19}\right)^2,\)

\( \union{ x = \displaystyle \frac{2 + \sqrt{19}}{3} \\ x = \displaystyle \frac{2 - \sqrt{19}}{3} }. \)

Сумма коэффициентов 0

Уравнение:

\(ax^2+bx + c = 0,~~a \neq 0,~~a+b+c=0.\)

Решение:

\( \union{ x = 1 \\ x = \frac{c}{a} }. \)

Пример:

\(3x^2 -7x+4 = 0 ,\)

\( \union{ x = 1 \\ x = \frac{4}{3} }. \)

Знакочередующаяся сумма коэффициентов 0

Уравнение:

\(ax^2+bx + c = 0,~~a \neq 0,~~a-b+c=0.\)

Решение:

\( \union{ x = -1 \\ x = -\frac{c}{a} }. \)

Пример:

\(3x^2 +7x+4 = 0 ,\)

\( \union{ x = -1 \\ x = -\frac{4}{3} }. \)

Первый коэффициент 1

Уравнение:

\(x^2+bx+c = 0.\)

Решение:

\(\cases{ x_1 \cdot x_2 = c \\ x_1+x_2=-b }~,\)

\( \union{ x = x_1 \\ x = x_2 }. \)

Пример:

\(x^2 -5x+6 = 0 ,\)

\( \cases{ x_1 \cdot x_2 = 6 \\ x_1 + x_2 = 5 } \Leftarrow \cases{ x_1 = 2 \\ x_2 = 3 }~, \)

\( \union{ x = 2 \\ x = 3 }. \)

Последний коэффициент \(\pm 1\)

Уравнение:

\(ax^2+bx \pm 1 = 0,~~a \neq 0.\)

Решение:

\({\tilde x}^2+b\tilde x \pm a = 0\),

\( \union{ \tilde{x} = \tilde{x_1} \\ \tilde{x} = \tilde{x_2} }, \)

\( \union{ x = \frac{\tilde x_1}{a} \\ x = \frac{\tilde x_2}{a} }. \)

Пример:

\(6x^2 +x-1 = 0 ,\)

\( {\tilde x}^2 + \tilde x - 6 = 0,\)

\( \union{ \tilde x = 2 \\ \tilde x = -3 }~, \)

\( \union{ x = \frac{2}{6} \\ x = \frac{-3}{6} }. \)
