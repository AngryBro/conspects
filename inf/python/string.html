Строка

  • Строка — упорядоченный набор символов.
  • Строка неизменяемый объект.
  • Нумерация символов начинается с 0.
  • Нумерация символов с конца строки начинается с -1.
Операция со строкой s Возвращаемое значение Сложность
s{{[{i}]}} Символ строки номер {i} \(O(1)\)
{t} {{in}} s Присутствует ли в строке {t} \(O(n)\)
{t} {{not in}} s Отсутствует ли в строке {t} \(O(n)\)
s{{[{i1}:{i2}:{step}]}} Строка из символов от {s[i1]} включительно,
до {s[i2]} не включительно, с шагом {step}		 \(O(|i1-i2|)\)
s{{[::-1]}} Строка в обратном порядке \(O(n)\)
s{{+}}{t} Строка дополненная с конца строкой {t} \(O(k)\)
{{len(}}s{{)}} Количество символов в строке \(O(1)\)
s{{.count({t})}} Количество {t} в строке \(O(n)\)
s{{.find({t})}} Индекс первого вхождения {t} в строку \(O(n)\)
s{{.rfind({t})}} Индекс последнего вхождения {t} в строку \(O(n)\)
s{{.replace({t1},{t2},{count})}} Строка с заменёнными {t1} на {t2},
заменяются первые {count} вхождений {t1} 		\(O(n)\)
s{{.split({t})}} Массив из частей строки, разделённой по разделителю {t} \(O(n)\)
s{{.splitlines()}} Массив из частей строки, разделённой по символу переноса строки \(O(n)\)
s{{.join({iterable})}} Соединённый {iterable} по строке \(O(n)\)
