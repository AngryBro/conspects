Массив (список)

  • Массив — упорядоченный набор объектов.
  • Нумерация объектов начинается с 0.
  • Нумерация объектов с конца массива начинается с -1.
Операция с массивом a Изменение массива Возвращаемое значение Сложность
a{{[{i}]}} нет Элемент массива номер {i} \(O(1)\)
{element} {{in}} a нет Присутствует ли в массиве {element} \(O(n)\)
{element} {{not in}} a нет Отсутствует ли в массиве {element} \(O(n)\)
a{{[}}{i1}{{:}}{i2}{{:}}{step}{{]}} нет Массив из элементов от {a[i1]} включительно,
до {a[i2]} не включительно, с шагом {step}		 \(O(|i1-i2|)\)
a{{[::-1]}} нет Массив в обратном порядке \(O(n)\)
a{{[{i}] = {element} }} Элемент номер {i} будет равен {element} нет \(O(1)\)
a{{+}}{b} нет Массив дополненный с конца массивом {b} \(O(k)\)
{{len(}}a{{)}} нет Количество элементов массива \(O(1)\)
{{sum(}}a{{)}} нет Сумма элементов массива \(O(n)\)
{{min(}}a{{)}} нет Минимальный элемент массива \(O(n)\)
{{max(}}a{{)}} нет Максимальный элемент массива \(O(n)\)
{{sorted(}}a{{,reverse={flag},key={function})}} нет Сортированный массив в заданном порядке,
по правилу {function}		 \(O(n\cdot \log{n})\)
{{list(set(}}a{{))}} нет Массив с уникальными элементами \(O(n)\)
a{{.copy()}} нет Копия массива \(O(n)\)
a{{.append({element})}} Добавляется {element} в конец массива нет \(O(1)\)
a{{.pop({i})}} Удаляется элемент номер {i} Удалённый элемент \(O(n-i)\)
a{{.count({element})}} нет Количество {element} в массиве \(O(n)\)
a{{.index({element})}} нет Номер {element} в массиве \(O(n)\)
a{{.sort(reverse={flag},key={function})}} Сортирует массив в заданном порядке,
по правилу {function}		 нет \(O(n\cdot \log{n})\)
