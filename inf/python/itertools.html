Itertools

  • Itertools — библиотека для работы с массивоподобными объектами.
  • Подключение: {{from}} itertools {{import}} *
Функция Возвращаемый массив кортежей
{{permutations({iterable},{length})}} Перестановки длиной {length}
{{product({iterable_1},{iterable_2},...,{iterable_n})}} Всевозможные комбинации, где:
  • Первый элемент один из {iterable_1}
  • Второй элемент один из {iterable_2}
  • ...
  • n-й элемент один из {iterable_n}
